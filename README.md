# Aula Cinco Git

![HTML](https://img.shields.io/badge/HTML5-estrutura-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![Git](https://img.shields.io/badge/Git-versionamento-F05032?style=for-the-badge&logo=git&logoColor=white)

Repositório de prática inicial com Git e HTML, usado para consolidar o fluxo básico de criação, alteração e versionamento de arquivos em um projeto web simples.

> Mesmo sendo um exercício introdutório, o foco está em registrar evolução técnica com organização, clareza e rastreabilidade.

## O que este projeto demonstra

| Competência | Evidência no repositório |
| --- | --- |
| Versionamento com Git | Projeto versionado em repositório público, com histórico de alterações |
| Estrutura HTML | Arquivo `index.html` como base de página estática |
| Disciplina de documentação | README objetivo explicando escopo, execução e valor do exercício |

## Estrutura

```text
.
├── README.md
└── index.html
```

## Como visualizar

```bash
git clone https://github.com/DouglasAntoni0/aula-cinco-git.git
cd aula-cinco-git
```

Depois, abra o arquivo `index.html` no navegador.

## Resultado técnico

Este projeto registra uma etapa de fundação: domínio do ciclo mínimo de trabalho com GitHub, edição de arquivos, organização de repositório e publicação de uma base HTML simples. Essa base sustenta práticas mais avançadas presentes nos projetos de automação e Quality Engineering deste perfil.
